## 🧬 Pilares do Paradigma 4.0

### 1. SNA Cognitivo (Semantic Network Analysis)

Diferente do RAG convencional que apenas recupera passagens de texto por similaridade vetorial, o CognixOS mapeia as **associações mentais** (Beliefs, Variables, Heuristics, Skills). Isso reconstrói o Léxico Mental do sujeito, permitindo prever vieses de pensamento e padrões comunicativos autênticos.

### 2. Soberania de IA: BYOK & Local First

O CognixOS 4.0 introduz a liberdade total de processamento:

* **BYOK (Bring Your Own Key)**: Conecte sua própria API Key do OpenRouter diretamente no painel, garantindo que você seja o dono dos seus créditos e dados.
* **IA Local (Ollama)**: Execute modelos 100% offline em sua máquina. O sistema roteia as requisições neuralmente para o seu localhost, garantindo privacidade absoluta.

### 3. Sincronização Física de Dados

Utilizando a **File System Access API**, o CognixOS rompe a sandbox do navegador para permitir que você salve seu histórico de chat diretamente no sistema de arquivos do seu computador (Desktop). Seus dados não precisam viver apenas no banco de dados; eles podem ser persistidos fisicamente sob seu controle direto.

### 4. Telegram Bridge (Omnichannel)

Conecte seus clones ao mundo real. Através de um sistema de vinculação via token UUID, você pode conversar com seus clones ativos diretamente pelo Telegram, enviando áudios e textos para processamento remoto.

## 🕵️ O Esquadrão (The Squad: 8-Core)

O processo de clonagem não é aleatório. Ele é executado por uma sequência determinística de 8 agentes especializados:

4. **Linguista**: Decodificação de sintaxe, n-grams e cadência rítmica.
5. **Cronista**: Tecelagem da Crônica de Identidade (Grafo SNA).
6. **Estrategista**: Forjamento de cenários de teste de estresse psicológico.
7. **Verificador**: Auditoria de integridade analítica e lógica epistêmica.
8. **Prompter**: Arquiteto final do DNA Neural (System Prompt).

## 🛠️ Instalação e Ativação

### 1. Pré-requisitos

* **Deno / Node.js** (Ambiente de execução)
* **Supabase CLI** (Para gerenciar o banco de dados e Edge Functions)
* **Ollama** (Opcional, para soberania local)

### 2. Setup Rápido

```bash
# Clone o ecossistema
git clone https://github.com/Joaofanis/CognixOS.git
cd CognixOS

# Instale as dependências
npm install

# Inicie o modo dev
npm run dev
```

### 3. Configuração do Backend

O CognixOS exige uma infraestrutura Supabase configurada com `pgvector` para a memória vetorial.
